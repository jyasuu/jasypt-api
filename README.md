# jasypt-api

```sh

curl -s "https://get.sdkman.io" | bash
source "$HOME/.sdkman/bin/sdkman-init.sh"
sdk install java 17.0.10-zulu
sdk install springboot
spring init jasypt-api --build=maven --java-version=17 --artifact-id jasypt-api --boot-version 3.2.2  --group-id com.jyasu --packaging=jar  --name jasypt-api
```
